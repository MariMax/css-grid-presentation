Grid lines

The vertical and horizontal lines that divide the grid and separate the columns and rows. first line right before grid container, last line right after container

Grid cell

A single unit of a CSS grid.

Grid area

Rectangular space surrounded by four grid lines. A grid area can contain any number of grid cells.

Grid track

The space between two grid lines. This space can be horizontal or vertical

Grid row

A horizontal track of a grid.

Grid column

A vertical track of a grid.

Gutter

The space between rows and columns in a grid. Grid container - The container that holds the entire CSS grid. It will be the element that has the display: grid or display: inline-grid property on it. Grid item - Any element that is a direct child of a grid container.
